# IPアドレス計算対策
CCNAのIPアドレス計算問題の例題を標準出力に、答えをテキストファイルに出力します。

# 使い方
`ruby subnet_calc.rb [問題数]`

例： `ruby subnet_calc.rb 20`

# 生み出されるもの
- terminal
```
$ ruby subnet_calc.rb 20
01: 51.161.4.139 /4
02: 122.122.30.80 /5
03: 60.109.110.63 /9
```

- ans.txt
```
1
48.0.0.0
63.255.255.255

2
120.0.0.0
127.255.255.255

3
60.0.0.0
```
